# AGS Volume API 设计 Cookbook

本目录是面向客户的 AGS Volume API 设计说明，仅包含文档材料。

中文 cookbook：[cookbook.zh-CN.md](cookbook.zh-CN.md)

## 内容

- 核心实体：`VolumeTemplate`、`VolumeMount`、`AgentCBS`、`Metadata`、`SandboxInstance.VolumeMounts`。
- COS、CFS、Agent Bucket 和 AGS 托管 AgentCBS 的典型使用方式。
- API 形态、请求字段、响应字段、约束和首期能力边界。
